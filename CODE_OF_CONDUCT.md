# Code of Conduct

## Expected behavior

People contributing to this project are expected to behave like adults.

That means:

- be respectful
- be specific
- criticize ideas, not people
- provide evidence when reporting bugs
- avoid abusive, insulting, or manipulative behavior
- do not post private data, credentials, or sensitive media

## Unacceptable behavior

The following is not acceptable:

- harassment
- personal attacks
- discriminatory language
- spam
- deliberate bad-faith disruption
- publishing private or sensitive material without permission

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block users when behavior damages the project or wastes maintainer time.

This is an engineering project, not a dumping ground for hostility.
