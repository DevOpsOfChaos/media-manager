# People review example artifacts

This directory contains non-sensitive, empty example JSON artifacts for the optional local people-review workflow.

The examples document the current machine-readable contracts for:

- `people-report.json` — scan summary and backend capability report
- `people-review-workflow.json` — editable review workflow input
- `people-review-workspace.json` — GUI/workbench-facing review workspace state

Real reports and review workspaces can reveal who appears in local files. Reports that include encodings can contain sensitive biometric metadata. Keep real generated files local and do not commit them.
