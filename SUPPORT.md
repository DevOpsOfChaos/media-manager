# Support

## What kind of support is reasonable right now

This project is in a pre-alpha stage. Expect rough edges, incomplete features, and breaking changes.

Reasonable support requests:

- reproducible bug reports
- setup problems with enough detail to reproduce them
- focused feature requests
- documentation gaps

Unreasonable support requests:

- vague complaints without reproduction steps
- requests to debug private media libraries without relevant logs
- broad "make it professional" requests with no concrete scope

## Before asking for help

Please include:

- operating system
- Python version
- whether ExifTool is installed
- the exact command or action you ran
- the full error message or traceback
- whether you used GUI or CLI

## Where to ask

- Bugs: open a GitHub issue with reproduction steps
- Feature ideas: open a focused GitHub issue
- Security issues: follow the instructions in `SECURITY.md`
